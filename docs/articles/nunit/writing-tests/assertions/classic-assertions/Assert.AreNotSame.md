# Assert.AreNotSame

**Assert.AreNotSame** tests that the two arguments do not reference the same object.

```csharp
Assert.AreNotSame(object expected, object actual);
Assert.AreNotSame(object expected, object actual,
                  string message, params object[] params);
```
